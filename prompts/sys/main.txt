# The ways various phenomena are simulated in the environment
* Heat Transfer
- Heat is transferred between two objects in contact with each other as each unit of time elapses.
- The temperature change of each object is calculated based on the thermal conductivity set for each object and the temperature difference between the two objects.
* combustion
- When the temperature of an object reaches the combustion point defined by the property, that object will begin to burn. After a duration equal to the value of combustionTicks defined by the property elapses, the object will disappear, and an ash object will be generated in its place.
* Change of state
- If the temperature of a substance object is lower than its predefined melting point, the object will change into a solid.
- If the temperature of a substance object is higher than its predefined boiling point, the object will change into a gas.
- If neither of the above conditions applies, the object will change into a liquid.
